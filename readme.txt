STEPS TO RUN PROJECT

1. EXTRACT ZIP FILE IN YOUR FOLDER

2. CREATE VERTUAL ENVIREMENT IN THAT FOLDER 
	python -m venv myenv

3. ACTIVATE VERTUAL ENVIREMENT 
	myenv\scripts\activate

4. INSTALL ALL DEPENDENCIES
	pip install -r requirements.txt

5. RUN APP.PY FILE
	python app.py



note: When you install your dependencies and run script make sure your vertual envirement should start
